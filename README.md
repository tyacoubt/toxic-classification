Compares three models: majority class baseline, NBOW + Logistic Regression, and DistilBERT.

## Design decisions

| Decision | Value | Rationale |
|---|---|---|
| Binary label | `max(subtypes, axis=1)` | Any toxicity subtype = toxic |
| Primary metric | F1 on toxic class | ~90% accuracy is achievable by predicting all non-toxic |
| Train / val / test | 80 / 10 / 10 stratified | Preserves class ratio across splits |
| Class imbalance — LR | `class_weight='balanced'` | Inverse-frequency weighting built into sklearn |
| Class imbalance — DistilBERT | Weighted `CrossEntropyLoss` | Applied during training only |
| DistilBERT `max_len` | 128 tokens | Covers ~95% of comments; see 01_eda.ipynb for justification |
| LR regularisation C | 1000 | Selected by 5-fold CV on val F1; plateau confirmed beyond C=10 |
| Random seed | 42 everywhere | Set in `src/data_utils.py` as `RANDOM_SEED` |

## Models

**Majority class baseline** — predicts non-toxic for every comment. ~90% accuracy, 0.0 F1 on toxic class. Proves that accuracy is a misleading metric on this dataset.

**NBOW + Logistic Regression** — TF-IDF (unigrams + bigrams, 50k features, sublinear TF) with logistic regression. C tuned by 5-fold CV. Runs entirely on CPU in under 5 minutes.

**DistilBERT** — `distilbert-base-uncased` fine-tuned for 3 epochs with AdamW (lr=2e-5) and weighted cross-entropy loss. Best checkpoint selected by validation F1. Needs a GPU — use Google Colab if training locally is too slow.

---

## Evaluation

All models are evaluated on the same held-out test split using F1 on the toxic class as the primary metric. Results are saved to `results/*.json` and compared in notebook 06.

## Error analysis

Categories are defined before any annotation begins. Two annotators label independently, Cohen's kappa is computed, and disagreements are resolved before analysis.

| Category | Definition |
|---|---|
| `sarcasm` | Toxic in intent, no overtly toxic surface words |
| `non_toxic_profanity` | Profanity used casually, not directed at anyone |
| `ambiguous` | Needs more context to determine toxicity |
| `discusses_hate` | Describes hateful content without being hateful |
| `implicit_toxicity` | Toxic through implication, no explicit slurs |
| `other` | Last resort |
